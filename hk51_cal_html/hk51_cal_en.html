HK Five One Calendar is created for Hong Kong BNO visa holder. The app records the dates of leave UK, and helps to calculate the allowable remaining number of days to leaeve UK for ILR and citizenship application.

Main Functions

  • Record BNO visa related dates
  • Calculate allowable days of leave UK for ILR and citizenship
  • Estimate ILR grant date
  • Estimate citizenship application date
  • Export data to csv
  • Use personal Google calendar for data storage, no 3rd party

User Guide

At the beginning, user has to login own Google account, The calendar is empty, and current month is shown.


After login, the app enters calendar view. Dates having event in the month shown are highlighted. Selecting any highlighted date shows the event details at the bottom.

At rigth side of month or event detail, there are two numbers shown. The top number shows the allowable days to leave UK for ILR requirement, and bottom number is for citizenship requirement. The numbers besides the month counts to the end of the month. Similarly, the numbers besides event detail counts to the end of the event.

Long pressing on event detail, a menu is shown for different operations.


List view shows the event in list format, the content differs according to data input.

OTher than the menu options in calendar view, user can jump to the month of an event selected.


The app enters event editing page when user select add or edit an event.

  • Firstly, select the event like to add at the top drop down menu. If editing event, there is no selectable events.
  • Most events need date selection only.
  • Visa approval has additional option of visa length.
  • For leave event, user can have description added and period of the leave. Leaving for one day is not useful in calculation, therefore the start and end date has to be different.

Settings page has options to change the appearnce, aand export data to csv.
